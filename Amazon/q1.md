- Problem Name : [Missing number in array](https://practice.geeksforgeeks.org/problems/missing-number-in-array1416/1)
- Platform : GFG

- Code:
```cpp

class Solution{
  public:
    int MissingNumber(vector<int>& array, int n) {
        // Your code goes here
         int sum = 0;
      for(int i =0;i<n-1;i++){
          sum = sum+array[i];
      }
      int total = ((n)*(n+1))/2;
      return total - sum;
    }
};
```
